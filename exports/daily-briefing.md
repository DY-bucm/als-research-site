# ALS 每日简报

更新时间：2026-06-26

1. **ALS中TDP-43病理的PBMC DEG/miRNA生物标志物**
   - 生物标志物 / 观察性临床研究 / high
   - 在ALS患者PBMC中鉴定出5个基因（Mctp1、Penk、Mt2A、Drd1、Rasgrp2）和4个miRNA组成的特征，能高准确度区分ALS与健康对照及PD，且与中枢TDP-43病理相关。
   - https://pubmed.ncbi.nlm.nih.gov/42251967/
2. **早发性神经炎症驱动由溶酶体PI(3,5)P₂不足引起的神经退行性变**
   - 临床试验 / 动物/细胞实验 / high
   - PI(3,5)P₂缺乏在神经退行性变发生前即引发早发性神经炎症，并伴随多种炎症性细胞死亡通路激活和蛋白质组发育重塑障碍。
   - https://pubmed.ncbi.nlm.nih.gov/42242586/
3. **IRE1通过核糖体相关质量控制调控ALS/FTD中TDP-43/TARDBP的蛋白稳态**
   - 临床试验 / 动物/细胞实验 / high
   - IRE1通过经典（转录激活Clbn/NEMF）和非经典（与Clbn/NEMF物理互作）两种途径激活RQC，降解异常TDP-43翻译产物，减轻蛋白毒性；在ALS小鼠模型中激活IRE1可改善病理和认知。
   - https://pubmed.ncbi.nlm.nih.gov/42341041/
4. **异构体特异性立体拉链驱动缩短型TDP-43的异常组装和错误定位**
   - 临床试验 / 观察性临床研究 / high
   - sTDP-43的羧基末端尾巴中的立体拉链簇驱动其聚集和胞质错误定位，而非核输出信号。
   - https://pubmed.ncbi.nlm.nih.gov/42341118/
5. **学习一种距离用于肌萎缩侧索硬化症患者的聚类分析**
   - 临床试验 / 临床试验研究 / high
   - 基于ALSFRS-R评分和弱监督学习，成功将ALS患者分为三个生存率显著不同的聚类，并确定了13个预测聚类归属的诊断参数。
   - https://pubmed.ncbi.nlm.nih.gov/42351201/
6. **静脉注射工程化AAV9基因沉默载体抑制人SOD1并延长ALS小鼠模型生存期**
   - 临床试验 / 动物/细胞实验 / high
   - 单次静脉注射工程化AAV9载体递送靶向SOD1的人工微RNA，可广泛抑制SOD1表达，保护运动神经元和神经肌肉接头，显著改善呼吸功能、运动表现并延长生存期，且在症状前和症状期给药均有效。
   - https://pubmed.ncbi.nlm.nih.gov/42350385/
7. **核凋亡介导蛋白毒性应激下的细胞死亡与神经退行性变**
   - 临床试验 / 观察性临床研究 / high
   - 核凋亡是蛋白毒性应激下一种由p38激酶调控的细胞死亡形式，在ALS/FTD模型和患者脑组织中存在。
   - https://pubmed.ncbi.nlm.nih.gov/42350373/
8. **通过调节自噬受体SQSTM1重塑ALS**
   - 治疗 / 待人工判定 / high
   - 哌唑嗪通过上调SQSTM1表达，在iPSC衍生运动神经元和斑马鱼模型中挽救SQSTM1单倍体不足导致的ALS表型。
   - https://pubmed.ncbi.nlm.nih.gov/42349421/
