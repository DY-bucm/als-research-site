# ALS 每日简报

更新时间：2026-06-24

1. **ALS中TDP-43病理的PBMC差异表达基因/miRNA生物标志物**
   - 生物标志物 / 观察性临床研究 / high
   - 在ALS患者PBMC中识别出5个差异表达基因和4个下调miRNA组成的特征，该特征在ALS中特异性失调，诊断AUC达0.87-1.00。
   - https://pubmed.ncbi.nlm.nih.gov/42251967/
2. **早发性神经炎症驱动由溶酶体PI(3,5)P2不足引起的神经退行性变**
   - 临床试验 / 动物/细胞实验 / high
   - 溶酶体PI(3,5)P2不足在神经退行性变发生前即引发早发性神经炎症，该炎症由小胶质细胞促氧化状态和多种炎症性细胞死亡通路介导，且部分由神经元和星形胶质细胞非细胞自主驱动。
   - https://pubmed.ncbi.nlm.nih.gov/42242586/
3. **肌萎缩侧索硬化症患者胃造口术后结局的多中心前瞻性评估**
   - 临床试验 / 观察性临床研究 / high
   - ALS患者胃造口术后超过半数仍持续体重下降，且与能量和蛋白质摄入不足相关；术后体重下降显著缩短生存期。
   - https://pubmed.ncbi.nlm.nih.gov/42336241/
4. **一种新出现的疾病相关运动神经元状态先于ALS中的细胞死亡**
   - 临床试验 / 动物/细胞实验 / high
   - 发现并定义了ALS中一种新的、先于细胞死亡的“疾病相关运动神经元”（DM）状态，该状态受特定转录因子网络调控，且在人类ALS中保守并与遗传风险变异相关。
   - https://pubmed.ncbi.nlm.nih.gov/42335888/
5. **邻近无序RGG结构域对FUS-RRM折叠态的稳定效应**
   - 临床试验 / 待人工判定 / high
   - 无序RGG结构域通过多价分子内接触稳定FUS RRM的折叠态，增加α1螺旋、β2、β3和KK环的结构比例。
   - https://pubmed.ncbi.nlm.nih.gov/42335378/
6. **口部运动皮层变薄与肌萎缩侧索硬化症言语障碍相关**
   - 临床试验 / 观察性临床研究 / high
   - 说话速度和发音速度降低与双侧口部运动皮层变薄相关，而ALSFRS-R延髓评分和神经学检查未显示这种关联。
   - https://pubmed.ncbi.nlm.nih.gov/42333954/
7. **从原发性侧索硬化修正诊断为肌萎缩侧索硬化：一项队列研究**
   - 临床试验 / 观察性临床研究 / high
   - 在PLS患者中，18%最终修正诊断为ALS；进展速度更快（SHR 3.08）和P-PLS诊断（vs D-PLS）是诊断修正的独立风险因素。
   - https://pubmed.ncbi.nlm.nih.gov/42214042/
8. **Target ALS全球自然史研究：跨平台蛋白质组学加速肌萎缩侧索硬化症生物流体标志物和药物靶点发现**
