# ALS 每日简报

更新时间：2026-06-24

1. **ALS中TDP-43病理的PBMC DEG/miRNA生物标志物**
   - 生物标志物 / 观察性临床研究 / high
   - 在ALS患者PBMC中鉴定出5个基因（Mctp1、Penk、Mt2A、Drd1、Rasgrp2）和4个miRNA组成的特征，能高精度区分ALS与健康对照及帕金森病（AUC 0.87-1.00）。
   - https://pubmed.ncbi.nlm.nih.gov/42251967/
2. **早发性神经炎症驱动由溶酶体PI(3,5)P2不足引起的神经退行性变**
   - 临床试验 / 动物/细胞实验 / high
   - 溶酶体PI(3,5)P2不足在神经退行性变发生前即引发早发性神经炎症，该炎症由小胶质细胞促氧化状态和神经元/星形胶质细胞非细胞自主信号驱动，并激活多种炎症性细胞死亡通路。
   - https://pubmed.ncbi.nlm.nih.gov/42242586/
3. **肌萎缩侧索硬化症众筹中未经证实的干细胞干预与补充和替代医学之间的联系**
   - 治疗 / 观察性临床研究 / high
   - ALS众筹活动中，对未经证实的干细胞干预的信心与对替代疗法和未经证实病因的请求高度相关，且疗效声明远超科学证据，导致患者面临财务和医疗风险。
   - https://pubmed.ncbi.nlm.nih.gov/42339807/
4. **Pridopidine 治疗肌萎缩侧索硬化症的 III 期疗效与安全性研究**
   - 临床试验 / 临床试验注册 / high
   - 尚未公布结果，主要终点为 ALSFRS-R 评分经死亡率校正后的变化。
   - https://clinicaltrials.gov/study/NCT07322003
5. **患者来源的肌萎缩侧索硬化模型是否成为新药发现的游戏规则改变者？**
   - 临床试验 / 待人工判定 / high
   - 患者来源的ALS模型目前不能作为独立决策工具，但可作为人类生物学过滤器用于靶点发现和患者分层。
   - https://pubmed.ncbi.nlm.nih.gov/42337904/
6. **肌萎缩侧索硬化症患者胃造口术后结局的多中心前瞻性评估**
   - 临床试验 / 观察性临床研究 / high
   - ALS患者胃造口术后体重持续下降普遍（51.6%），且与能量和蛋白质摄入不足相关，术后体重下降显著缩短生存期。
   - https://pubmed.ncbi.nlm.nih.gov/42336241/
7. **一种新出现的疾病相关运动神经元状态在ALS中先于细胞死亡**
   - 临床试验 / 动物/细胞实验 / high
   - 识别出一种名为“疾病相关运动神经元”（DMs）的独特细胞状态，该状态在α运动神经元死亡前出现，其分子特征在人类ALS脊髓中保守，且相关基因组区域富集ALS遗传风险变异。
   - https://pubmed.ncbi.nlm.nih.gov/42335888/
8. **邻近无序RGG结构域对FUS-RRM折叠态的稳定效应**
   - 临床试验 / 待人工判定 / high
   - 无序RGG结构域通过多价分子内接触网络稳定FUS RRM的折叠态，增加α1螺旋、β2、β3和KK环的结构比例。
   - https://pubmed.ncbi.nlm.nih.gov/42335378/
