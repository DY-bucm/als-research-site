3. **准备肌萎缩侧索硬化症诊所，为携带ALS风险变异的个体提供纵向护理**
   - 遗传 / 观察性临床研究 / high
   - 到2035年，美国无症状ALS基因携带者数量将达2.6万，多数州ALS诊所年就诊需求将超过100次，需显著扩展基础设施。
   - https://pubmed.ncbi.nlm.nih.gov/42367369/
4. **ALS/FTD发病机制中的先天免疫串扰**
   - 临床试验 / 综述/荟萃分析 / high
   - ALS/FTD中先天免疫通路（cGAS-STING、NLRP3、TREM2-DAP12）被致病蛋白激活并相互串扰，驱动疾病进展。
   - https://pubmed.ncbi.nlm.nih.gov/42359357/
5. **ALS中TDP-43病理的PBMC DEG/miRNA生物标志物**
   - 生物标志物 / 观察性临床研究 / high
   - 在ALS患者PBMC中鉴定出5个基因（Mctp1、Penk、Mt2A、Drd1、Rasgrp2）和4个miRNA组成的特征，能高精度区分ALS与健康对照及帕金森病（AUC 0.87-1.00）。
   - https://pubmed.ncbi.nlm.nih.gov/42251967/
6. **早发性神经炎症驱动由溶酶体PI(3,5)P₂不足引起的神经退行性变**
   - 临床试验 / 动物/细胞实验 / high
   - PI(3,5)P₂不足在神经退行性变发生前即引发早期神经炎症，且该炎症是疾病起始的关键驱动因素。
   - https://pubmed.ncbi.nlm.nih.gov/42242586/
7. **鞘内注射利鲁唑治疗肌萎缩侧索硬化症**
   - 临床试验 / 随机对照试验 / high
   - 鞘内持续输注利鲁唑（最大4.8 mg/d）在两例ALS患者中耐受超过两年，未出现运动或感觉并发症，且无口服常见的乏力副作用。
   - https://pubmed.ncbi.nlm.nih.gov/42263252/
