1. **ALS中TDP-43病理的PBMC差异表达基因/miRNA生物标志物**
   - 生物标志物 / 观察性临床研究 / high
   - 在ALS患者PBMC中识别出5个差异表达基因（Mctp1、Penk、Mt2A、Drd1、Rasgrp2）和4个下调miRNA，构成高诊断准确性（AUC 0.87-1.00）的疾病特异性特征。
   - https://pubmed.ncbi.nlm.nih.gov/42251967/
2. **早发性神经炎症驱动由溶酶体PI(3,5)P₂不足引起的神经退行性变**
   - 临床试验 / 动物/细胞实验 / high
   - 溶酶体PI(3,5)P₂不足在神经退行性变发生前即引发早期神经炎症，该炎症是疾病起始的关键驱动因素，而非单纯后果。
   - https://pubmed.ncbi.nlm.nih.gov/42242586/
3. **从原发性侧索硬化修正诊断为肌萎缩侧索硬化：一项队列研究**
   - 临床试验 / 观察性临床研究 / high
   - 在PLS患者中，18%最终修正诊断为ALS；更快的进展速度（ALS功能评定量表斜率）和可能PLS（P-PLS）诊断是修正的独立风险因素。
   - https://pubmed.ncbi.nlm.nih.gov/42214042/
4. **神经退行性疾病中的微量元素稳态失衡与有毒金属神经毒性**
   - 临床试验 / 临床试验研究 / high
   - 金属稳态失衡在神经退行性疾病中发挥多重作用，铁螯合治疗在临床试验中虽减少脑铁但恶化临床结局，提示需要重新评估金属靶向策略。
   - https://pubmed.ncbi.nlm.nih.gov/42332177/
5. **肌肉超声是ALS中敏感的结果测量指标**
   - 临床试验 / 临床试验研究 / high
   - 肌肉超声回声强度在3-12个月内对ALS进展的效应量（1.09）高于ALSFRS-R（0.84）和CMAP（0.77），是更敏感的结构生物标志物。
   - https://pubmed.ncbi.nlm.nih.gov/42324866/
6. **赞助基因检测对170名连续同意的肌萎缩侧索硬化症患者的影响：单中心回顾性研究**
   - 治疗 / 观察性临床研究 / high
   - 在170名ALS患者中，赞助基因检测的致病性变异检出率为12.9%，其中家族性ALS为37.1%，散发性ALS为6.7%；C9orf72和SOD1是最常见的致病基因；检测结果直接指导了遗传咨询和SOD1靶向治疗。
   - https://pubmed.ncbi.nlm.nih.gov/42324839/
7. **一例ALS患者上运动神经元兴奋性变化的直接证据**
