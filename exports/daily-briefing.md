1. **ALS中TDP-43病理的PBMC差异表达基因/miRNA生物标志物**
   - 生物标志物 / 观察性临床研究 / high
   - 在ALS患者PBMC中鉴定出五个差异表达基因（Mctp1、Penk、Mt2A、Drd1、Rasgrp2）和四个下调miRNA，构成一个能高准确度区分ALS与健康对照及帕金森病的诊断特征。
   - https://pubmed.ncbi.nlm.nih.gov/42251967/
2. **早发性神经炎症驱动由溶酶体PI(3,5)P₂不足引起的神经退行性变**
   - 临床试验 / 动物/细胞实验 / high
   - 溶酶体PI(3,5)P₂缺乏在神经退行性变发生前即引发以补体激活和免疫细胞浸润为特征的早发性神经炎症，该炎症是神经退行性变的起始驱动因素。
   - https://pubmed.ncbi.nlm.nih.gov/42242586/
3. **从原发性侧索硬化修正诊断为肌萎缩侧索硬化：一项队列研究**
   - 临床试验 / 观察性临床研究 / high
   - 18%的PLS患者在病程4年内修正诊断为ALS，进展速度（ALSFRS-R斜率）是独立预测因子（SHR 3.08）。
   - https://pubmed.ncbi.nlm.nih.gov/42214042/
4. **神经退行性疾病中的微量元素稳态失衡与有毒金属神经毒性**
   - 临床试验 / 临床试验研究 / high
   - 金属稳态失衡在神经退行性疾病中发挥核心作用，但铁螯合治疗（去铁酮）在减少脑铁的同时却恶化临床结局，提示现有金属靶向策略需要机制性重新评估。
   - https://pubmed.ncbi.nlm.nih.gov/42332177/
5. **肌肉超声是ALS中敏感的结果测量指标**
   - 临床试验 / 临床试验研究 / high
   - 肌肉超声回声强度在3-12个月内比ALSFRS-R和CMAP更敏感地检测ALS进展，效应量最大（1.09），且通过发病特异性肌肉选择可进一步提高反应性。
   - https://pubmed.ncbi.nlm.nih.gov/42324866/
6. **ALS患者上运动神经元兴奋性变化的直接证据**
   - 机制 / 机制研究 / high
   - ALS患者中，直接激活皮质脊髓神经元反应减弱，但跨突触激活反应增强，且短间隔皮层内抑制（SICI）减少，表明上运动神经元过度兴奋源于谷氨酸能输入增强和抑制减弱。
   - https://pubmed.ncbi.nlm.nih.gov/42324254/
