# ALS 每日简报

更新时间：2026-06-25

1. **ALS中TDP-43病理的PBMC DEG/miRNA生物标志物**
   - 生物标志物 / 观察性临床研究 / high
   - 在ALS患者PBMC中鉴定出5个基因（Mctp1、Penk、Mt2A、Drd1、Rasgrp2）和4个miRNA组成的特征，该特征能高精度区分ALS与健康对照及PD（AUC 0.87-1.00），且与中枢TDP-43病理相关。
   - https://pubmed.ncbi.nlm.nih.gov/42251967/
2. **早发性神经炎症驱动由溶酶体PI(3,5)P₂不足引起的神经退行性变**
   - 临床试验 / 动物/细胞实验 / high
   - 溶酶体PI(3,5)P₂不足在神经退行性变发生前即引发早发性神经炎症，该炎症由小胶质细胞、神经元和星形胶质细胞共同驱动，并涉及多种炎症性细胞死亡通路。
   - https://pubmed.ncbi.nlm.nih.gov/42242586/
3. **IRE1通过核糖体相关质量控制调控ALS/FTD中TDP-43/TARDBP的蛋白稳态**
   - 临床试验 / 动物/细胞实验 / high
   - IRE1通过经典和非经典途径激活RQC通路，促进异常TDP-43翻译产物的降解，从而减轻TDP-43病理并改善ALS小鼠模型的认知功能。
   - https://pubmed.ncbi.nlm.nih.gov/42341041/
4. **异构体特异性立体拉链驱动缩短型TDP-43的异常组装和错误定位**
   - 临床试验 / 观察性临床研究 / high
   - sTDP-43的聚集和错误定位由其羧基末端尾巴中的立体拉链驱动，而非核输出信号。
   - https://pubmed.ncbi.nlm.nih.gov/42341118/
5. **未经证实的干细胞干预与补充替代医学在肌萎缩侧索硬化众筹中的关联**
   - 治疗 / 观察性临床研究 / high
   - ALS众筹活动中，对未经证实的干细胞疗法的信心与对补充替代医学的请求高度相关，且活动发起者普遍夸大疗效，导致患者和捐赠者面临财务和医疗风险。
   - https://pubmed.ncbi.nlm.nih.gov/42339807/

7. **患者来源的肌萎缩侧索硬化模型是否是新药发现中的游戏规则改变者？**
   - 临床试验 / 待人工判定 / high

   - https://pubmed.ncbi.nlm.nih.gov/42337904/
8. **肌萎缩侧索硬化症患者胃造口术后结局的多中心前瞻性评估**
